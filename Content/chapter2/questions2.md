 1. ### What is the purpose of the “Hello, World!” program?
 To give an introduction and demonstrate some beginner concepts such as printing to the screen and introducing the main() function.

 2. ### Name the four parts of a function.
 return type, function name, parameter list, function body

 3. ### Name a function that must appear in every C++ program.
 int main() {}

 4. ### In the “Hello, World!” program, what is the purpose of the line return 0;?
 We don't actually use the return value, but on Linux for example it can be used to determine if the program finished successfully (zero meaning success).

 5. ### What is the purpose of the compiler?
 Translate human readable code into machine code.

 6. ### What is the purpose of the #include directive?
 To include files into other files, most often header files.

 7. ### What does a .h suffix at the end of a file name signify in C++?
 Header file that contains definitions of terms.

 8. ### What does the linker do for your program?
 Takes the object code and links together the different translation units to create the executable.

 9. ### What is the difference between a source file and an object file?
 A source file contains human readable code and an object file is the result of source files being compiled and it is machine code.

 10. ### What is an IDE and what does it do for you?
 Interactive development environment and it provides useful features for programming like syntax highlighting or a GUI for programming instead of doing it through a command line and

 11. ### If you understand everything in the textbook, why is it necessary to practice?
 To solidify concepts and because learning cannot happen just by observing, you need to practice things yourself to truly be able to apply them.
